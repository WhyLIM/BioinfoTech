安装 R 包​

参考：

  • loomR 包的官方文档
  • monocle3 包的官方文档

依赖安装​

sudo apt install libfontconfig1-dev libcurl3-dev libxml2-dev libssl-dev cmake libgit2-dev libharfbuzz-dev libfribidi-dev libtiff-dev libhdf5-dev libcairo2-dev libxt-dev libgdal-dev libudunits2-dev libicu-dev

R 包安装​

loomR​

# Install devtools from CRAN
install.packages("devtools")
# Use devtools to install hdf5r and loomR from GitHub
devtools::install_github(repo = "hhoeflin/hdf5r")
devtools::install_github(repo = "mojaveazure/loomR", ref = "develop")

Seurat​

BiocManager::install("Seurat")

monocle3​

if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install(version = "3.17")

注意 BiocManager 3.17 适用于 R 4.3.0，R 4.2.x 请安装 BiocManager 3.16 ，其他参考 这里

不会自动安装的依赖：

BiocManager::install(c('BiocGenerics', 'DelayedArray', 'DelayedMatrixStats',
                       'limma', 'lme4', 'S4Vectors', 'SingleCellExperiment',
                       'SummarizedExperiment', 'batchelor', 'HDF5Array',
                       'terra', 'ggrastr', 'spdep'))

speedglm 这个包源目前（2023-04-27）有一点问题，可以在 此处 下载 Package source (tar.gz) ，手动在 RStudio 中安装

devtools::install_github('cole-trapnell-lab/monocle3')

了解 loom 文件​

与标准的 R 对象将其中包含的所有数据加载到内存中不同，loom 对象只是连接到磁盘上的一个文件，这使得它能够以低内存读取大规模的数据集

展示 loom 文件

> lfile
Class: loom
Filename: /home/mli/Study/DeTumor/GSE156793_S3_gene_count.loom
Access type: H5F_ACC_RDWR
Attributes: CreationDate, LOOM_SPEC_VERSION, last_modified, version
Listing:
       name    obj_type    dataset.dims dataset.type_class
  col_attrs   H5I_GROUP            <NA>               <NA>
 col_graphs   H5I_GROUP            <NA>               <NA>
     layers   H5I_GROUP            <NA>               <NA>
     matrix H5I_DATASET 4062980 x 63561          H5T_FLOAT
  row_attrs   H5I_GROUP            <NA>               <NA>
 row_graphs   H5I_GROUP            <NA>               <NA>

需要注意的是，不同的 loom 文件可能会使用不同的对象名存储细胞名称信息，因此需要根据具体情况来确定正确的对象名。可以使用 ls() 函数列出 loom 文件中所有的对象，然后查找你需要的对象名称。

ls(lfile[["col_attrs"]])

这个命令输出了名为 lfile[["col_attrs"]] 的对象中可用的函数和方法列表，其中包括：

你可以使用以下命令列出所有的列属性对象：

names(lfile[["col_attrs"]])

lfile[["col_attrs"]] 表示 lfile 对象中的 col_attrs 组，col_attrs 组中包含了许多列属性（column attributes），比如细胞名称、实验批次、样本组等信息。这些列属性在 loom 文件中使用 HDF5 格式存储。

> names(lfile[["col_attrs"]])
 [1] "All_reads"           "Assay"               "Batch"               "Development_day"     "Exon_reads"         
 [6] "Experiment_batch"    "Fetus_id"            "Intron_reads"        "Main_cluster_name"   "Main_cluster_umap_1"
[11] "Main_cluster_umap_2" "Organ"               "Organ_cell_lineage"  "RT_group"            "Sex"                
[16] "Size_Factor"         "batch"               "obs_names"           "sample"

这些属性包括：

  • All_reads：该细胞的总读数。
  • Assay：Assay 的名称。
  • Batch：表示该细胞所属的批次。
  • Development_day：样本采集的开发天数。
  • Exon_reads：该细胞的外显子读数。
  • Experiment_batch：表示该实验所属的批次。
  • Fetus_id：胎儿的 ID。
  • Intron_reads：该细胞的内含子读数。
  • Main_cluster_name：该细胞所属的主要簇名称。
  • Main_cluster_umap_1：该细胞在 UMAP 图中的 x 坐标。
  • Main_cluster_umap_2：该细胞在 UMAP 图中的 y 坐标。
  • Organ：该细胞所属的组织器官。
  • Organ_cell_lineage：该细胞所属的器官细胞系谱。
  • RT_group：该细胞所属的 RT 组。
  • Sex：该细胞所属个体的性别。
  • Size_Factor：归一化的大小因子。
  • batch：样品所属批次。
  • obs_names：该列对应的观测名称。
  • sample：样品的名称。

如果要查看 sample 的信息，运行

lfile[["col_attrs/sample"]]

或者

lfile$col.attrs$sample

输出如下

> lfile[["col_attrs/sample"]]

而如果要看具体内容，可以在后面加上索引/切片

lfile[["col_attrs/sample"]][1:10]

如果嫌一个个看麻烦，那么可以写个循环查看各组的第一个数据

for (i in names(lfile[["col_attrs"]])) {
  group <- paste0("col_attrs/", i)
  print(paste(i, lfile[[group]][1], sep = ": "))
}

同样，可以使用以下命令列出所有的行属性对象：

names(lfile[["row_attrs"]])

得到：

> names(lfile[["row_attrs"]])
[1] "exon_intron"     "gene_id"         "gene_short_name" "gene_type"       "index"           "var_names"

这些属性包括：

  • exon_intron: 提供了基因是外显子还是内含子的信息。
  • gene_id: 提供了基因的唯一标识符。
  • gene_short_name: 提供了基因的常用名称。
  • gene_type: 提供了基因的类型，例如编码蛋白质、非编码 RNA 等。
  • index: 提供了行索引。
  • var_names: 提供了行的名称。
