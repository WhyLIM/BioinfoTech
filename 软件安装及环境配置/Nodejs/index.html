Node.js

安装 Node.js​

Nodejs 的安装可以参考 这个仓库

以下是一个简单的示例：

安装 nvm​

nvm 是一个用于管理多个 Node.js 版本的工具，它可以让你在同一台机器上同时使用不同版本的 Node.js，方便开发和测试。

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash

添加环境变量​

以上代码会自动添加环境变量到 ~/.bashrc，如果没有，手动添加

sudo gedit ~/.bashrc

写入

export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] && printf %s "${HOME}/.nvm" || printf %s "${XDG_CONFIG_HOME}/nvm")"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh" # This loads nvm

保存并关闭，刷新

source ~/.bashrc

验证 nvm 安装​

command -v nvm

输出 nvm 则已经安装成功

安装 node​

安装 node，目前（2023-04-25）的最新稳定版本是 18.16.0 Hydrogen

如何查看：

nvm ls-remote

如何安装：

nvm install 18.16.0

验证 node 安装​

node -v
