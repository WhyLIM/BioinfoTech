通过 docker 使用微信、TIM

安装 docker​

sudo apt update
sudo apt install curl
sudo curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun

配置国内镜像源​

新建或修改文件 daemon.json

sudo gedit /etc/docker/daemon.json

写入内容

{
  "registry-mirrors" : [
    "http://ovfftd6p.mirror.aliyuncs.com",
    "http://registry.docker-cn.com",
    "http://docker.mirrors.ustc.edu.cn",
    "http://hub-mirror.c.163.com"
  ],
  "insecure-registries" : [
    "registry.docker-cn.com",
    "docker.mirrors.ustc.edu.cn"
  ],
  "debug" : true,
  "experimental" : true
}

保存并重启 docker 服务，确定是否换源成功，如果如下命令能看到 daemon.json 里的网址表示换源成功

sudo systemctl daemon-reload
sudo systemctl restart docker
sudo docker info

拉取微信和 TIM 的镜像​

# 微信
sudo docker pull bestwu/wechat
# TIM
sudo docker pull bestwu/qq:office

安装微信​

容器生成​

此处注意输入法框架，我的是 IBus

# 微信
sudo docker run -d --name wechat --device /dev/snd --ipc=host \
-v /tmp/.X11-unix:/tmp/.X11-unix \
-v $HOME/wechat/WeChatFiles:/WeChatFiles \
-e DISPLAY=unix$DISPLAY \
-e XMODIFIERS=@im=IBus \
-e QT_IM_MODULE=IBus \
-e GTK_IM_MODULE=IBus \
-e AUDIO_GID=`getent group audio | cut -d: -f3` \
-e GID=`id -g` \
-e UID=`id -u` \
bestwu/wechat

启动容器​

# 微信
sudo docker start wechat

安装 TIM​

容器生成​

# TIM
sudo docker run -d --name tim --device /dev/snd --ipc=host \
-v $HOME/tim/TencentFiles:/TencentFiles \
    -v /tmp/.X11-unix:/tmp/.X11-unix \
-e XMODIFIERS=@im=IBus \
-e QT_IM_MODULE=IBus \
-e GTK_IM_MODULE=IBus \
-e DISPLAY=unix$DISPLAY \
-e AUDIO_GID=`getent group audio | cut -d: -f3` \
-e VIDEO_GID=`getent group video | cut -d: -f3` \
-e GID=`id -g` \
-e UID=`id -u` \
bestwu/qq:office

启动容器​

# TIM
sudo docker start tim

配置快捷方式​

配置当前用户 sudo 无需密码​

sudo gedit /etc/sudoers

添加内容

# xxx 是当前用户名
xxx ALL=NOPASSWD: ALL

添加启动脚本​

将自定义图标（ico,png etc.）添加到对应文件夹里

微信脚本​

sudo mkdir ~/tencent/wechat
sudo gedit ~/tencent/wechat/wechat.sh

脚本内容

#!/bin/sh
sudo docker restart wechat

TIM 脚本​

sudo mkdir ~/tencent/tim
sudo gedit ~/tencent/tim/tim.sh

脚本内容

#!/bin/sh
sudo docker restart tim

赋予执行权限​

sudo chmod +x ~/tencent/wechat/wechat.sh
sudo chmod +x ~/tencent/tim/tim.sh

创建快捷方式​

sudo gedit /usr/share/applications/wechat.desktop
sudo gedit /usr/share/applications/tim.desktop

分别写入内容

微信​

[Desktop Entry]
Version=1.0
Name=WeChat
Exec=/home/mli/tencent/wechat/wechat.sh
Icon=/home/mli/tencent/wechat/wechat.ico
Type=Application
Encoding=UTF-8
Terminal=false

TIM​

[Desktop Entry]
Version=1.0
Name=TIM
Exec=/home/mli/tencent/tim/tim.sh
Icon=/home/mli/tencent/tim/tim.ico
Type=Application
Encoding=UTF-8
Terminal=false

第一次启动会慢一些，耐心等待。

关于更新​

两者的版本都比较老，微信版本为 2.6.8.65，是 docker 下兼容的最后一个版本；

而 TIM 可以手动更新，封装的老版本 TIM 只能扫码登录，更新方法：

# 启动 TIM，窗口启动后不要关闭，否则 docker 会自动停止 
sudo docker start tim
sudo docker exec -it tim bash
# 下载 wget 并下载新版本 TIM
apt update
apt install wget
cd /tmp
wget https://dldir1.qq.com/qqfile/qq/TIM3.4.6/TIM3.4.6.22074.exe
https://dldir1.qq.com/qqfile/qq/TIM3.4.5/TIM3.4.5.22071.exe -O tim.exe
su qq
WINEPREFIX="/home/qq/.deepinwine/Deepin-TIM" deepin-wine tim.exe

需要注意的是有些按钮可能不会显示，需要自行想象

# 安装 OpenGL 库文件
sudo apt install mesa-utils
sudo apt install libgl1-mesa-dev
